# GUI DASHBOARD

The GUI Dashboard is the user interface for the mF2C agent. Its main page includes links for:
* CIMI Resources Browser
* SLA Violations
* User Management
* Service LC and instances

## Usage

Browse to https://\<agent>/dahsboard/
